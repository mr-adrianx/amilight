# amilight
Lightweight basic MiLigh Linux-script with some support for RGB, RGBW, and WWCW bulbs.


May work on some of these: AppLamp, MiLight, LimitlessLED, Applight, Applamp, LEDme, dekolight, iLight, Easybulb
